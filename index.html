Help & Documentation

Everything you need to go from a stack of PNGs to a multi-color 3D print.

Jump to: What is this? What you need Workflow Start Size Settings Layers Generate Slicer FAQ

What is this?

PNG23D turns flat images into a multi-color 3D-printable object — most often a business card, but any flat print works. You upload one or more transparent-background PNGs. Each PNG becomes a layer in the print, with its own color and its own thickness, stacked like slices of a cake.

The app hands you back a .3mf file that any modern slicer can open and print.

Privacy: Everything runs in your browser. No files are uploaded to a server. No account. Close the tab and nothing is kept except the files you downloaded.

What you need

  • Transparent-background PNGs — one per color. SVGs also work.
  • A slicer that opens .3mf — free options: Bambu Studio, OrcaSlicer, PrusaSlicer. Simplify3D also works.
  • A 3D printer. Multi-color systems (Bambu AMS, Prusa MMU, Mosaic Palette) print this automatically. Single-color printers can still do it — the slicer inserts a pause at each color change and prompts you to swap filament.

The workflow

  • 1. Start — name the project.
  • 2. Size & DPI — tell the app how big the finished print should be in mm.
  • 3. Print Settings — pick your nozzle and layer height.
  • 4. Arrange Layers — upload a PNG for each layer, set color and thickness, reorder as needed.
  • 5. Generate — click the button, download the .3mf (print file) and .3dp (project save).
  • 6. Slice & print — open the .3mf in your slicer, assign filament colors, print.

Page 1 — Start

  • Project name — required. Drives the downloaded filenames (e.g. SpencerCard_20260419_143012.3mf). Keep it short and descriptive.
  • New Project — starts a blank project with zero layers.
  • Drop .3dp Project File — reload a previously saved project. Every PNG and setting comes back exactly as you left it.

Page 2 — Size & DPI

This is the real-world physical size of the finished print, in millimeters.

  • Presets — US business card (85.725 × 53.975mm), 50×50, 100×100, 235×235mm (full Ender-3 bed).
  • Card width / height — type exact numbers in mm. The default is a landscape US business card.
  • Lock Proportions — keeps the aspect ratio when you edit one dimension.
  • Reference PNG + DPI — drop a PNG here and enter its DPI (300 is typical for print artwork). The app calculates physical width and height from the pixel dimensions. Use this when you've designed artwork at a known print resolution.

Page 3 — Print Settings

  • Nozzle diameter — match your printer's physical nozzle. Most stock printers are 0.4mm. Smaller nozzles capture more detail but print slower.
  • Layer height — defaults to half the nozzle (0.2mm for a 0.4mm nozzle). Lower = smoother vertical walls, more print layers, longer print.
  • Alpha threshold (0–255) — the cutoff that decides whether a pixel counts as "on." 128 (50% opacity) is the default. Lower it to keep faint / feathered edges; raise it to clip them cleanly.
  • Quality (10–100%) — how finely the PNG is voxelized before meshing. 100% uses every pixel; 50% is ~4× lighter (quality has a quadratic effect on triangle count). Drop this while iterating, bump it up for final export.

Page 4 — Arrange Layers

The core step. Each card in the stack is one layer of the print.

  • Top of list = top of the print. Drag cards by their body to reorder.
  • Upload artwork — drop a PNG (or SVG) into the card's dropzone. Transparent pixels = empty space; opaque pixels = filled with this layer's color.
  • Name — cosmetic label that appears in the slicer object list.
  • Color — cosmetic in the 3MF; most slicers respect it when auto-assigning filament.
  • Thickness — set by mm (physical) or layers (count of print layers). Changing one updates the other automatically. A typical base layer is 0.8mm; accent layers 0.2–0.4mm.
  • Flip X / Y (⇆ / ⇅) — mirror the layer horizontally or vertically. Needed for double-sided prints so the back face reads correctly.
  • + Inlay — optional second PNG that cuts through the base layer at the same height, filled with a different color. Think of a colored pad with a logo inset into it.
  • Trash icon — removes the layer.
  • Live thumbnail — shows the current layer + inlay composite. Updates as you change artwork, color, or flip toggles.
  • 3D preview — drag to orbit, drag down to tilt under the model. Auto-fits to whatever you've built so far.

Generate & download

Clicking Generate produces two files, downloaded together:

  • projectname_YYYYMMDD_HHMMSS.3mf — the print file. Open this in your slicer.
  • projectname_YYYYMMDD_HHMMSS.3dp — the saved project. Drop it onto Page 1 later to edit or hand to a collaborator.

A progress bar shows which layer is building plus an estimated time remaining. Complex designs at 100% quality can take a minute or two — the preview stays responsive while it works.

In your slicer

  • Open the .3mf. You'll see one object per layer, already stacked at the correct heights.
  • Assign a filament color to each object. In Bambu Studio / Orca this is the paint-bucket icon or right-click → "Change filament."
  • Slice and print. On single-color printers, the slicer will insert a pause at each color change — swap filament when prompted and resume.
  • On multi-material printers (AMS, MMU, Palette), color changes happen automatically.

FAQ

Do I need a multi-color printer?
No. Any FDM printer works. Single-color printers pause at each color boundary and prompt a filament swap — the slicer handles this for you.
What slicer should I use?
Anything that opens .3mf. Bambu Studio and OrcaSlicer are the easiest for multi-color assignment; PrusaSlicer and Simplify3D also work.
What resolution does my PNG need to be?
Minimum ~214px wide (0.4mm per pixel at card width); recommended 857px+ (0.1mm per pixel). Lower-res PNGs look blocky when printed at millimeter scale.
Can I use SVGs?
Yes. The browser rasterizes them to a canvas at your selected quality, the same way it handles PNGs.
My PNG has soft edges and the print looks chunky.
Lower the alpha threshold on Page 3 to keep more of the feathered edge. Or raise it if you want edges to clip cleanly. Also make sure the source PNG is high enough resolution — soft edges on a low-res file become pixel steps.
What is an inlay?
A second shape that punches through a layer at the same height, filled with a different color. Example: a pink pad with a black logo cut into it — the logo is the inlay. Use "+ Inlay" on a layer card to add one.
What's the smallest detail that will print cleanly?
Roughly 1.2mm (about 3× a 0.4mm nozzle width). Anything thinner tends to break off or not extrude reliably. Smaller nozzles can go finer but print slower.
Is anything I upload sent to a server?
No. The entire app runs in your browser. Close the tab and your files are gone except for the .3mf and .3dp you downloaded.
Browser stalled or ran out of memory.
Drop the Quality slider on Page 3. Quality has a quadratic effect on mesh size — 50% quality is ~4× lighter than 100%. Very large meshes (>2M triangles) can stall a tab.
Why is my .3mf so big?
Every opaque pixel becomes voxel geometry. Reduce Quality, simplify the artwork (fewer fine details), or raise the alpha threshold to eliminate near-transparent pixels.
My text prints backwards / my logo is mirrored.
Use the Flip X or Flip Y toggle on that layer's card. Common fix for double-sided prints where the back face is the mirror image of the front.
What's a .3dp file?
A save file. JSON with all your settings and every uploaded PNG packed in. Drop one onto Page 1 to reload the project exactly as it was.
Can I share a .3dp with someone else?
Yes. They'll get the exact same project, artwork included. It's a complete, self-contained file.
The 3D preview says THREE is unavailable.
The preview uses the Three.js CDN. If that fails to load (offline, blocked, or CDN issue) the preview disables but upload, generate, and download all still work normally.
Can I run this offline?
Yes — save index.html to disk and open it. The only feature that requires the internet is the 3D preview (Three.js loads from a CDN).
Can I print this on a resin printer?
Technically yes — it's flat extruded geometry. But this app is tuned for multi-color FDM. On resin, color is irrelevant and you'd only get the combined silhouette.
What card orientation is the default?
Landscape: 85.725mm wide × 53.975mm tall (3.375" × 2.125" — standard US business card). Change it on Page 2 if you want a different orientation or size.
How many layers can I stack?
No hard limit — practical ceiling is driven by browser memory and total print height. Two to five layers covers almost every real-world business-card design.
